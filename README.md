# Tamales Website

A mock website that sells tamales. [View Live](https://tamales-web.netlify.app/).

## Technologies Used
- ReactJS
- CSS3
- React Slick

## Features
- Mobile-first approach
- React Slick's carousel
- CSS Grid
- Responsiveness
